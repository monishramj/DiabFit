DiabFit is an (currently Android only) Flutter app that calculates personalized mealtime insulin doses through a three-step wizard: food tracking, medical info input, and blood glucose monitoring. Integrates with Edamam API for real-time nutritional data. Please note that DiabFit does not constitute medical advice; for any inquiries, consult your healthcare provider.

## Tech Stack

```
Framework: Flutter 3.16+ (Dart 3.2.3+)  
API: Edamam Food Database v2 (REST)  
UI: Material Design 3, Custom animations  
```
